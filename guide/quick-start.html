# 快速开始

# 1. clone项目到本地

git clone https://github.com/geekidea/spring-boot-plus.git

# 2. maven构建

默认使用local环境,对应配置文件：application-local.yml

mvn clean package -Plocal

# 3. 新建数据库

默认使用MySQL数据库

  • 默认数据库名称为：spring_boot_plus
  • 导入表结构和数据：
spring-boot-plus/docs/db/mysql_spring_boot_plus.sql

👉Download mysql_spring_boot_plus.sql

# 4. 启动redis

redis-server

# 5. 检查是否安装lombok插件

如遇到log报错，则需要安装lombok插件

# 👉 IDEA导入spring-boot-plus

# 👉 Eclipse中使用spring-boot-plus

# 6. 启动项目

项目启动入口类

SpringBootPlusApplication.java

# 7. 访问Spring Boot Admin

👉 http://localhost:8888

  • spring-boot-plus主界面，Spring Boot Admin

# 8. 访问Swagger

👉 http://localhost:8888/swagger-ui.html

  • Swagger界面

在 GitHub 上编辑此页
