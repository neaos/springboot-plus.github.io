# 图中红框出，是项目信息

  • projectFinalName：项目名称
  • projectVersion：项目版本
  • profileActive：当前maven profile激活环境
  • 例如local，对应配置文件application-local.yml配置
  • contextPath：项目访问根路径
  • port：项目端口
  • home：项目主界面，Spring Boot Admin监控界面
  • docs：swagger接口界面
